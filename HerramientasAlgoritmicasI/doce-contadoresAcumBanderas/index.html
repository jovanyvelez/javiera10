🚪 El Contador

⏱️ 45 minutos · "El Portero del Club" — cuenta de uno en uno

🧠 La idea central

Imagina que eres el portero de un club y tienes un clic-counter manual (esos aparatitos que hacen "clic" cada vez que entra alguien). Cada vez que pasa una persona, aprietas el botón y el número sube en 1. Nunca sube en 2, ni en 5. Siempre de 1 en 1. Eso es un contador.

Un contador es una variable (normalmente int) que se inicializa en 0 y se incrementa de a 1 en 1 cada vez que ocurre un evento que quieres contar.

📝 Las dos formas de escribirlo

✨ Forma corta (profesional)

int contador = 0; // inicialización // Dentro de un bucle o evento: contador++; // suma 1

📝 Forma explícita

int contador = 0; // inicialización // Dentro de un bucle o evento: contador = contador + 1; // suma 1

💡 ¿Son iguales?

Sí, hacen exactamente lo mismo. Pero contador++ es la forma que usan los programadores profesionales. También existe contador += 1, que es una versión intermedia. Las tres son válidas.

PorteroClub.cs
using System;

class PorteroClub
{
    static void Main()
    {
        // ══════════════════════════════════════
        // 1. DECLARAR E INICIALIZAR EL CONTADOR
        // ══════════════════════════════════════
        int contador = 0;           // Empieza en CERO
        const int AFORO = 100;      // Capacidad máxima

        Console.WriteLine("🎉 Bienvenido al Club Neón");
        Console.WriteLine($"Aforo máximo: {AFORO} personas\n");

        // ══════════════════════════════════════
        // 2. SIMULAR LA NOCHE (bucle)
        // ══════════════════════════════════════
        bool clubAbierto = true;

        while (clubAbierto && contador < AFORO)
        {
            Console.Write("¿Entra alguien? (s/n): ");
            string respuesta = Console.ReadLine();

            if (respuesta == "s")
            {
                contador++;   // ← AQUÍ ESTÁ EL CONTADOR
                Console.WriteLine($"🚶 Persona #{contador} entró. " +
                    $"Quedan {AFORO - contador} lugares.");
            }
            else if (respuesta == "n")
            {
                clubAbierto = false;
            }
        }

        // ══════════════════════════════════════
        // 3. MOSTRAR EL RESULTADO FINAL
        // ══════════════════════════════════════
        Console.WriteLine($"\n🎊 Noche terminada.");
        Console.WriteLine($"Total de clientes: {contador}");
    }
}

🛒 El Acumulador

⏱️ 45 minutos · "El Carrito de Compras" — suma valores variables

🐷 Analogía: La alcancía mágica

Imagina una alcancía donde metes monedas de diferentes valores: a veces $5, a veces $20, a veces $1. Cada vez que metes una moneda, se suma al total que ya tenías. El contador siempre suma 1; el acumulador suma lo que tú le eches.

🧠 La diferencia clave con el contador

🚪 Contador

Siempre suma 1
contador++;
contador += 1;

🛒 Acumulador

Suma valores variables
total += precio;
suma = suma + valor;

📝 Las dos formas de escribirlo

✨ Forma corta

double total = 0; // inicia en 0 // Dentro del bucle: total += precio; // suma el precio

📝 Forma explícita

double total = 0; // inicia en 0 // Dentro del bucle: total = total + precio; // suma el precio

💻 Código completo en C#

TiendaGadgets.cs
using System;

class TiendaGadgets
{
    static void Main()
    {
        // ══════════════════════════════════════
        // 1. INICIALIZAR ACUMULADOR Y CONTADOR
        // ══════════════════════════════════════
        double total = 0;       // Acumulador de dinero
        int contador = 0;       // Contador de productos

        Console.WriteLine("🛒 Bienvenido a GadgetStore");
        Console.WriteLine("Escribe 'fin' para terminar la compra\n");

        // ══════════════════════════════════════
        // 2. BUCLE DE COMPRA
        // ══════════════════════════════════════
        while (true)
        {
            Console.Write("Precio del producto (o 'fin'): ");
            string entrada = Console.ReadLine();

            if (entrada.ToLower() == "fin")
                break;   // Sale del bucle

            // Validar que sea un número
            if (double.TryParse(entrada, out double precio) && precio > 0)
            {
                total += precio;    // ← ACUMULADOR
                contador++;         // ← CONTADOR

                Console.WriteLine($"✓ Añadido. " +
                    $"Productos: {contador} | Total: ${total:F2}\n");
            }
            else
            {
                Console.WriteLine("⚠️ Ingresa un número válido.\n");
            }
        }

        // ══════════════════════════════════════
        // 3. CALCULAR PROMEDIO (usa ambos)
        // ══════════════════════════════════════
        Console.WriteLine("\n════ RESUMEN DE COMPRA ════");
        Console.WriteLine($"Productos comprados: {contador}");
        Console.WriteLine($"Total pagado: ${total:F2}");

        if (contador > 0)
        {
            double promedio = total / contador;   // ← usa ambos
            Console.WriteLine($"Precio promedio: ${promedio:F2}");
        }
    }
}

🚨 La Bandera

⏱️ 45 minutos · "El Sensor de Seguridad" — ON/OFF, verdadero/falso

💡 Analogía: El interruptor de luz

Un interruptor solo tiene dos estados: encendido (ON) o apagado (OFF). No hay intermedio. Una bandera (flag en inglés) funciona igual: es una variable bool que vale true o false, y sirve para recordar si algo pasó.

🧠 ¿Cuándo usar una bandera?

Usa una bandera cuando necesites responder preguntas del tipo "¿pasó X?":

  • ¿Se encontró el producto en la lista?
  • ¿Hubo algún error en los datos?
  • ¿Algún estudiante reprobó?
  • ¿El usuario ya está registrado?

📝 Las dos partes de una bandera

🏁 Declarar bandera

// Empieza "apagada" (no ha pasado nada) bool encontrado = false; bool hayError = false; bool esVIP = false;

🔥 Activar bandera

// Cuando ocurre el evento: encontrado = true; // ← ¡se activó! // Y luego puedes consultarla: if (encontrado) { /* ... */ }

💻 Código completo en C#

EscánerSeguridad.cs
using System;

class EscánerSeguridad
{
    static void Main()
    {
        // Lista de productos (algunos prohibidos)
        string[] productos = {
            "Laptop", "Libro", "Cuchillo",   // ← prohibido
            "Cámara", "Botella agua", "Tijeras"  // ← prohibido
        };

        string[] prohibidos = { "Cuchillo", "Tijeras", "Arma", "Explosivo" };

        // ══════════════════════════════════════
        // 1. INICIALIZAR BANDERA (apagada)
        // ══════════════════════════════════════
        bool alarma = false;
        string itemDetectado = "";
        int revisados = 0;

        Console.WriteLine("🔍 Iniciando escaneo...\n");

        // ══════════════════════════════════════
        // 2. REVISAR CADA PRODUCTO
        // ══════════════════════════════════════
        foreach (string producto in productos)
        {
            revisados++;

            // ¿Está en la lista de prohibidos?
            bool esProhibido = false;
            foreach (string p in prohibidos)
            {
                if (producto == p)
                {
                    esProhibido = true;
                    break;
                }
            }

            if (esProhibido)
            {
                alarma = true;          // ← ¡BANDERA ACTIVADA!
                itemDetectado = producto;
                Console.WriteLine($"🚨 ALERTA en item #{revisados}: {producto}");
                break;   // Ya no seguimos buscando
            }
            else
            {
                Console.WriteLine($"✓ Item #{revisados}: {producto} — OK");
            }
        }

        // ══════════════════════════════════════
        // 3. REVISAR LA BANDERA AL FINAL
        // ══════════════════════════════════════
        Console.WriteLine("\n════ REPORTE FINAL ════");
        Console.WriteLine($"Productos revisados: {revisados}");

        if (alarma)   // ← CONSULTAMOS LA BANDERA
        {
            Console.WriteLine($"🚨 ALARMA ACTIVADA");
            Console.WriteLine($"Item prohibido detectado: {itemDetectado}");
            Console.WriteLine($"Acción: Llamar a seguridad");
        }
        else
        {
            Console.WriteLine($"✅ Todo limpio. Pasajero autorizado.");
        }
    }
}

⚠️ Nombres que comunican intención

Las banderas deben tener nombres que empiecen con verbos o preguntas: encontrado, esVIP, hayError, aprobado. Nunca nombres como x, temp o b. Tu "yo" del futuro te lo agradecerá.

💻 El código completo: las 3 herramientas juntas

CajeroNeon.cs
using System;

class CajeroNeon
{
    static void Main()
    {
        Console.WriteLine("💰 CAJERO NEÓN — Sistema de cobro v1.0\n");

        // ══════════════════════════════════════════════════════
        // 1. INICIALIZAR LAS 3 HERRAMIENTAS
        // ══════════════════════════════════════════════════════
        int contador = 0;           // Cuántos productos
        double total = 0;           // Cuánto dinero
        bool esVIP = false;         // ¿Supera los $100?

        const double LIMITE_VIP = 100.0;
        const double DESCUENTO_VIP = 0.10;   // 10%

        // ══════════════════════════════════════════════════════
        // 2. BUCLE PRINCIPAL: leer productos
        // ══════════════════════════════════════════════════════
        string seguir = "s";

        while (seguir == "s")
        {
            Console.Write("Nombre del producto: ");
            string nombre = Console.ReadLine();

            Console.Write("Precio: $");
            double precio = double.Parse(Console.ReadLine());

            // AQUÍ TRABAJAN CONTADOR Y ACUMULADOR JUNTOS
            contador++;         // ← Contador: suma 1
            total += precio;    // ← Acumulador: suma el precio

            Console.WriteLine($"✓ {nombre} añadido. " +
                $"Subtotal: ${total:F2}\n");

            Console.Write("¿Otro producto? (s/n): ");
            seguir = Console.ReadLine();
        }

        // ══════════════════════════════════════════════════════
        // 3. EVALUAR LA BANDERA
        // ══════════════════════════════════════════════════════
        if (total > LIMITE_VIP)
        {
            esVIP = true;    // ← BANDERA se activa
        }

        // ══════════════════════════════════════════════════════
        // 4. IMPRIMIR TICKET (usa las 3 variables)
        // ══════════════════════════════════════════════════════
        Console.WriteLine("\n════ TICKET DE VENTA ════");
        Console.WriteLine($"Productos: {contador}");
        Console.WriteLine($"Subtotal:  ${total:F2}");

        if (contador > 0)
        {
            double promedio = total / contador;   // ← usa ambas
            Console.WriteLine($"Promedio:  ${promedio:F2}");
        }

        if (esVIP)   // ← BANDERA se consulta aquí
        {
            double descuento = total * DESCUENTO_VIP;
            double final = total - descuento;

            Console.WriteLine($"\n⭐ ¡COMPRA VIP!");
            Console.WriteLine($"Descuento 10%: -${descuento:F2}");
            Console.WriteLine($"TOTAL FINAL:   ${final:F2}");
        }
        else
        {
            Console.WriteLine($"\nTOTAL FINAL:   ${total:F2}");
        }

        Console.WriteLine("\n¡Gracias por su compra! 🎉");
    }
}
